Política de Privacidad

Esta página describe el tratamiento de datos en este sitio académico de demostración. No se recopila información personal real a través de esta web.

  1. Este sitio es una maqueta de proyecto y no utiliza bases de datos de usuarios.
  2. Los mensajes enviados desde el correo electrónico se gestionan solo para contestar dudas.
  3. No se comparte información con terceros.

En caso de contactar por correo, los datos recibidos se usan únicamente para responder y no se almacenan de forma automatizada.
